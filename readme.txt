### ðŸŒŸ PLUGIN FEATURES

* Fully customizable list items with built-in easy-to-use content creator.
* Supports Meta Mask, Binance Wallet and countless other wallets that are compatible with Wallet Connect.
* Maximize user engagement with customizable voting icons : heart, thumbs up, up/down voting.
* Create any type of lists - plain text, paragraphs with text and images, bulleted/numbered lists, videos etc.
* Option to reorder you list based on the ranking.
* Lots of other features - show/hide vote count, change position of the voting icon etc.
* Use on-the-go! Our plugin is 100% Mobile friendly and reliable.
* Get started right away with our easy-to-use style settings.
* Compatible with Elementor plugin - use the poll short-code in your Elementor post creator.
